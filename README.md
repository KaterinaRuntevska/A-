# Апликација за управување со студенти

Оваа JavaScript апликација симулира префрлање на студенти од редовни во вонредни студии по одредени предмети.

## Функционалности:
* Дефинирана е функција `vonredni()` која го намалува бројот на студенти за 5.
* Се врши автоматска проверка на три предмети:
  1. Структурно програмирање (110 студенти)
  2. Веб програмирање (50 студенти)
  3. ООП (5 студенти)
* Ако по префрлањето бројот на студенти за одреден предмет е 0, програмата испишува известување дека сите студенти се префрлени.

## Како се користи:
Потребно е да имате инсталирано [Node.js](https://nodejs.org/). Стартувајте ја програмата со следнава команда во терминал:
```bash
node index.js
